# 🐍 贪吃蛇

经典贪吃蛇网页游戏，纯前端实现，无需安装依赖，打开即玩。

## 🎮 玩法

| 操作 | 按键 |
|------|------|
| 移动 | `↑ ↓ ← →` 或 `W A S D` |
| 暂停/继续 | `空格` 或点击按钮 |
| 开始 | 点击「开始游戏」 |

> 📱 手机端会自动显示虚拟方向键

## ✨ 特性

- 暗色主题，带发光效果
- 吃到食物后速度逐渐加快
- 记录单局最高分
- 碰墙或咬到自己则游戏结束

## 🚀 运行

直接用浏览器打开 `index.html` 即可：

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

## 📁 结构

```
snake/
└── index.html   # 游戏全部代码（HTML + CSS + JS）
```
